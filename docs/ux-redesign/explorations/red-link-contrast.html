Why the glow halo matters

  • The original "thin red text" feeling comes from too few colored pixels per glyph at 13px Antonio. The contrast ratio is fine; the eye just doesn't have enough red mass to latch onto.
  • Three small dials beat one big change:
    • Weight 600 -> 700: ~12–15% more pixel mass per stroke.
    • Size 13px -> 14px: ~16% area bump per glyph.
    • LED glow halo: the text-shadow bloom adds ~2–3px of red haze around each letter, which the eye reads as "this red is brighter / more present" without changing the actual hue.
  • The glow is also on-brand: it matches the LED bloom already used on the brand mark, focus rings, and active stage badge. The link starts to feel like a piece of instrument hardware, not an underlined hyperlink.
  • If even this feels like "too much", Option C alone (brighter LED-soft red on plain Antonio) is the lowest-effort fix: it bumps contrast to 6.51:1 (AAA-ish) by changing exactly one hex value. Hue shifts toward pink slightly.
